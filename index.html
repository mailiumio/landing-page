Simple email marketing for static sites

Mailium makes it easy to handle newsletter form submissions on static sites. It’s an API driven email marketing platform.

The goal is to take away the pain of working with existing email marketing platforms as a developer. You get to control all markup and styling. Titanium takes care of the rest.

This means:

  • Vue, React and JS libraries
  • Thorough documentation
  • Code examples and integration guides
  • Error handling
  • No embedded forms
